# 📚 Books Scraper & Data Analysis Project

## 📌 Overview

This project is a complete **data pipeline** that scrapes book data from an e-commerce website and performs data cleaning, analysis, and visualization.

It demonstrates how raw data can be transformed into meaningful insights.

---

## 🚀 Features

* 🔍 Web scraping using BeautifulSoup
* 🧹 Data cleaning and preprocessing using Pandas
* 📊 Exploratory Data Analysis (EDA)
* 📈 Data visualization using Matplotlib
* ⚙️ Automated workflow using main.py

---

## 🛠️ Tech Stack

* Python
* Requests
* BeautifulSoup
* Pandas
* Matplotlib

## ▶️ How to Run

### Step 1: Clone the repository

```
git clone https://github.com/Krishna-vamsee/books-scraper-project.git
cd books-scraper-project
```

### Step 2: Create virtual environment

```
python -m venv venv
venv\Scripts\activate
```

### Step 3: Install dependencies

```
pip install -r requirements.txt
```

### Step 4: Run the project

```
python main.py
```
