#  Fast Food Consumption & Health Impact Analysis

##  Overview
This project analyzes the impact of fast food consumption on overall health using data science and machine learning techniques. It aims to identify patterns and build predictive models for health outcomes.

---

##  Objectives
- Perform Exploratory Data Analysis (EDA)
- Analyze relationship between fast food consumption and health
- Predict overall health score (Regression)
- Classify health levels (Low, Medium, High)
- Compare multiple machine learning models

---

##  Dataset Features
- Age  
- Gender  
- BMI  
- Fast Food Consumption Frequency  
- Lifestyle Habits  
- Health Indicators  

---

##  Tech Stack
